# Moins.ui | UI/UX & Product Design Portfolio

## Project Overview
This repository serves as a professional documentation mirror and backup for the **Moins.ui** portfolio website. The site is built to demonstrate industry-standard UI/UX design, focusing on high-quality product experiences for SaaS teams. It aligns with a strategic digital roadmap aimed at delivering clean, accessible, and performant web experiences.

**Live Site:** [https://milo90542-tevgj.wordpress.com/](https://milo90542-tevgj.wordpress.com/)

---

## Technical Specifications
To meet the requirements of the Digital Roadmap, the following technical choices were made:

- **Platform:** WordPress.com (Free Version)
- **Editor:** Gutenberg / WordPress Block Editor
- **PHP Version:** 8.0+ (Platform default)
- **Theme:** Minimalist, mobile-responsive block theme focused on visual hierarchy.
- **SSL/Security:** Active HTTPS via WordPress.com global security layer.

## Implementation Details & "The How"
### 1. Visual Hierarchy & UX
- Custom layouts were built using the Block Editor to establish a clear F-pattern reading flow.
- Strategic placement of high-fidelity SaaS dashboard case studies to establish immediate visual authority.
- Integrated social proof through a dedicated Testimonials section.

### 2. Manual SEO Optimization
As this project utilizes the free version of WordPress, SEO was optimized manually without the use of plugins:
- **Permalinks:** Manually adjusted page slugs (e.g., changed `/writing/` to `/case-studies/`) for better keyword relevance.
- **Alt Text:** Every project image and UI mockup contains descriptive, keyword-rich Alt Text for accessibility and image search optimization.
- **Headings:** Strict adherence to H1-H3 hierarchy to ensure search engine crawlers understand content priority.
- **Internal Linking:** Cross-linked the 'About' and 'Services' pages to improve site crawlability and user flow.

### 3. Performance & Responsiveness
- **Image Optimization:** All UI/UX mockups were compressed prior to upload to ensure fast loading times without a caching plugin.
- **Breakpoints:** Verified layout integrity across mobile, tablet, and desktop devices to ensure "zero design breaking."
- **Accessibility:** Achieved a high Accessibility score (98/100) on PageSpeed Insights. The site is fully navigable via keyboard (Tab key).

## Plugin Manifest
While the free version of WordPress.com handles most functionality natively, the following features are active:
- **Jetpack:** Core performance and security features.
- **WP.com Global CDN:** Automatic image optimization and delivery.
